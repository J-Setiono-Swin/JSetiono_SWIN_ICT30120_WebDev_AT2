This website is a simple demonstration of HTML markup, CSS style formatting, JavaScript functions, and file/folder organisation. 
  The website contains a few GIF images which have been tool-tipped and given alt text to provide better accessibility for screen readers/the visually impaired. 
  There is also a navigation bar styled with CSS that contains links to the three different pages on the site. 

Two JavaScript features have been used. 
  1.	In the about page, a button is used to change paragraph text into a different sentence.
  2.	In the contacts page, a button is used to show a GIF and a line of text that is initially hidden upon opening the page. 

The website has been tested on Firefox and Chromium (ver.151.0.7912.0). 
  All functions work as intended, and there are no formatting/layout issues between the two.
